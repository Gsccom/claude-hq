# Test Checklist — Beyoncé Rule
# "If you liked it then you shoulda put a test on it"

## 서버 기동 테스트
- [ ] 포트 비어있음: lsof -ti:7778 = 없음
- [ ] npm run dev 오류 없이 시작
- [ ] curl HTTP 200: curl -o /dev/null -w "%{http_code}" http://localhost:7778/
- [ ] 모든 라우트 200 응답

## UI 시각 테스트
- [ ] 헤더 + 지휘체계 배너 표시
- [ ] KJV 말씀 표시
- [ ] 다크 배경 (#060a14) 적용
- [ ] 모든 셀렉트박스 동작
- [ ] 반응형 레이아웃 (모바일 확인)

## Claude API 스트리밍 테스트
- [ ] 메시지 전송 → 스트리밍 응답
- [ ] API 키 없을 때 안내 메시지 표시 (에러 화면 아님)
- [ ] 빈 입력 시 전송 버튼 비활성화
- [ ] 긴 응답 스크롤 자동

## AI 패널 테스트
- [ ] 생성 시작 → 7개 패널 동시 활성화
- [ ] 진척도바 애니메이션 동작
- [ ] 로그 라인 순차 추가
- [ ] 완료 시 ✅ + 상태 변경
